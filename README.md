syUtils 个人编写收集的工具类，以及封装了AFNetwork网络请求库的使用
===========================
引入:syUtilsHeader.h即可使用

目录结构如下：
------Foundation Category 存储封装好的Foundation类操作   
------UIKit Category      控件操作Category   
------Fuzzy               动态模糊处理工具类   
------Macro               常用宏封装   
------NetWork             网络工具类的封装
